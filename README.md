# Dinamic-site

Practice project: building a dynamic website from scratch.

## Technologies

- HTML5
- CSS 3
- Bootstrap 5.3
- JavaScript

## Features

- Responsive header with Bootstrap grid
- Dropdown submenu on hover (pure CSS)
- Carousel slider

## How to run

Open Dinamic_site/index.html in your browser.
